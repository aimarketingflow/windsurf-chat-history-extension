üì¶ What Gets Captured

‚úÖ Captures:

  • Currently open/visible chat windows
  • Active chat sessions (what's on screen)
  • Chat panels rendered in the DOM
  • All messages in visible chats

‚ùå Does NOT Capture:

  • Closed chat sessions (not visible)
  • Historical chats (from Windsurf's database)
  • Other users' chats (only yours)
  • Chats not currently loaded in browser
Important: The script reads from the browser DOM (what's currently rendered), not from Windsurf's internal database. Make sure to open all chats you want to backup before running the script!

üí° Best Practices

Daily Workflow:

Morning: Start Windsurf, open yesterday's chats, press Cmd+Shift+B
During Work: Backup after important conversations
End of Day: Open today's chats, press Cmd+Shift+B

Before Important Work:

  1. Open all relevant project chats
  2. Run backup (Cmd+Shift+B)
  3. Work on project
  4. Run backup again before closing

Weekly Archive:

  1. Open all chats from this week
  2. Run backup
  3. Move file to archive folder
  4. Repeat next week

üîß Troubleshooting

Extension Not Working:

  • Reload Windsurf: Cmd+Shift+P ‚Üí "Reload Window"
  • Check Extensions panel - is it enabled?
  • Try reinstalling the extension

Desktop Shortcut Fails:

  • Check accessibility permissions: System Preferences ‚Üí Security & Privacy ‚Üí Accessibility
  • Make sure Windsurf is running first
  • Try the V2 version: üéØ Start Windsurf + Capture V2.command

No Chats Captured:

  • Make sure chat windows are open and visible
  • Try switching between chats to load them
  • Run showAvailableChats() in DevTools to see what's available

Keyboard Shortcut Conflict:

  • Open: Preferences ‚Üí Keyboard Shortcuts (Cmd+K Cmd+S)
  • Search: windsurf-chat-backup
  • Change to a different shortcut
