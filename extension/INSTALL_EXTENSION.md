# Install Windsurf Chat Backup Extension

## Quick Install Steps

1. Open Windsurf
2. Press Cmd+Shift+P
3. Type: "Developer: Install Extension from Location"
4. Navigate to: `~/windsurf-chat-history-extension/extension/`
5. Select the folder
6. Reload Windsurf

Press `Cmd+Shift+B` to backup your chat!
