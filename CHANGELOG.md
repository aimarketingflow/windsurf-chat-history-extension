## [1.0.0] - 2025-11-03

### Added
- Initial release of Windsurf Chat Backup Toolkit
- Windsurf extension with keyboard shortcut (Cmd+Shift+B)
- Desktop launcher for one-click backups
- Manual backup script for DevTools
- Beautiful HTML documentation guide
- Automatic installer script
- MIT License
- Comprehensive README
- Contributing guidelines

### Features
- 100% local processing (no cloud uploads)
- Multiple backup methods (extension, desktop, manual)
- JSON output format
- Auto-save to Downloads folder
- Keyboard shortcut support
- Command palette integration
- Detailed logging
- Error handling and troubleshooting

### Documentation
- Installation guide
- User guide (HTML)
- Troubleshooting guide
- API reference
- Contributing guidelines

### Security
- No external API calls
- No telemetry or tracking
- Local-only data processing
- Open source and auditable

## Release Notes

### v1.0.0 - Initial Release

This is the first public release of the Windsurf Chat Backup Toolkit!

**What's Included:**
- ✅ Windsurf extension
- ✅ Desktop shortcut
- ✅ Manual backup script
- ✅ HTML documentation
- ✅ Auto-installer

**Platform Support:**
- ✅ macOS (tested on macOS 14+)
- ⏳ Windows (coming soon)
- ⏳ Linux (coming soon)

**Known Limitations:**
- Only captures visible/open chats
- Cannot access Windsurf's internal database
- macOS only (uses AppleScript)

**Getting Started:**
1. Download the release
2. Run `./install.sh`
3. Press `Cmd+Shift+B` to backup!

**Feedback Welcome:**
This is the first release! Please report any issues or suggestions on GitHub.
